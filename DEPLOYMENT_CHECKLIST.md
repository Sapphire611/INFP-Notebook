## 部署步骤

### 1. 数据库部署
```bash
# 在 Supabase SQL Editor 中运行
psql -f supabase/add-mbti-field.sql
```

### 2. 后端部署
```bash
cd /Users/liuliyi/code/INFP-cms
npm run build
# 部署到生产服务器
```

### 3. 小程序部署
```bash
cd /Users/liuliyi/code/INFP-Notebook
npm run build:mp-weixin
# 在微信开发者工具中上传代码
# 提交审核
```
